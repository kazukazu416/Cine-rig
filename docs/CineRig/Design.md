## デザインフィロソフィー

### 基本思想
Appleの「引き算の美学」を、プロツールに適用する。
参考: Logic Pro / Final Cut Pro / DaVinci Resolve

### タイポグラフィ
- フォント: SF Pro Display / Inter / -apple-system
- 階層は太さ（weight）で表現（300/400/500/600）
- サイズは控えめ（11〜14pxが基本）

### 余白
- 要素間に十分な余白を取る
- 詰め込みすぎない（プロ密度は保ちつつ）

### カラー
- ベース: 明るいニュートラル（#FAFAFA〜#FFFFFF 系）
- アクセント: 深いマリンブルー（#005BA6）の1色のみ
- 彩度は抑えめ（鮮やかにしない）

### 角丸
- すべての要素に統一的な角丸（6〜10px）
- ボタン・カード・入力欄すべて

### ボーダー・影
- 1px の極細ライン（rgba 0.08〜0.15）
- 影は極力使わない、使うなら 0 2px 8px rgba(0,0,0,0.04) 程度

### インタラクション
- ホバー: 背景色がわずかに変化
- トランジション: 200〜300ms ease-out
- 急な変化を避ける

### アイコン
- SF Symbols 風（lucide-react など線画系）
- 単色 / 1.5px ストローク

### NG事項
- グラデーション多用
- 強い影（ドロップシャドウ）
- 派手なアニメーション
- 角ばったボタン
- 太いボーダー

## カラーシステム

### アクセントカラー（マリンブルー）

| 用途 | 値 |
|------|----|
| メイン | `#005BA6` |
| ホバー | `#0070C9` |
| 押下時 | `#004A88` |
| 薄い背景 | `#E6F0FA` |

**使用箇所**: 主要CTAボタン・アクティブ状態・選択中・リンク・強調

### ベース背景

| 用途 | 値 |
|------|----|
| ページ背景 | `#FAFAFA` |
| カード / パネル | `#FFFFFF` |
| セカンダリ背景 | `#F5F5F7` |
| ホバー背景 | `#F0F0F2` |

### ボーダー

| 用途 | 値 |
|------|----|
| 標準 | `rgba(0,0,0,0.08)` |
| 強調 | `rgba(0,0,0,0.15)` |

### テキスト

| 用途 | 値 |
|------|----|
| メイン | `#1d1d1f` |
| セカンダリ | `#6e6e73` |
| 補助 | `#86868b` |

### 配線図固有

- ノード背景: `#FFFFFF`
- キャンバス背景: `#FAFAFA`

## 参考UIの方向性

### 全体構成
- 上部タブ（Structure / Data / Access / Settings 的な切替）
- 左サイド：機能・機材カテゴリ＋検索
- メインキャンバス：ノードベース配線図
- 右サイド：縦アイコンメニュー（パネル切替）
- 右下：ミニマップ

### カラー運用
- 背景は明るめ（白〜薄グレー `#F5F5F7` 系）
- アクセントカラーは `#005BA6` の1色のみ
- ノードのカテゴリ色は天面の細いバーのみ

### ノード
- コンパクトサイズ（情報密度高め）
- 角丸控えめ（4〜6px）
- 影は極めて繊細（ほぼフラット）
- 左右にハンドル（接続点）が明示

### タイポ
- ノード内テキスト：12〜13px
- ラベル：11〜12px
- 階層は太さで（500/600）

### NG
- 派手なグラデーション
- 強い影
- 装飾的なアイコン
- 大きなボタン

### 推奨
- フラットデザイン
- 高情報密度
- 明るいベースカラー
- 単色アクセント（マリンブルー `#005BA6`）
