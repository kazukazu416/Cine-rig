## デザインフィロソフィー

### 基本思想
Appleの「引き算の美学」を、プロツールに適用する。
参考: Logic Pro / Final Cut Pro / DaVinci Resolve

### タイポグラフィ
- フォント: SF Pro Display / Inter / -apple-system
- 階層は太さ（weight）で表現（300/400/500/600）
- サイズは控えめ（11〜14pxが基本）

### 余白
- 要素間に十分な余白を取る
- 詰め込みすぎない（プロ密度は保ちつつ）

### カラー
- ベース: ニュートラルグレー（#1a1a1a 〜 #f5f5f7 系）
- アクセント: 各カテゴリ1色のみ
- 彩度は抑えめ（鮮やかにしない）

### 角丸
- すべての要素に統一的な角丸（6〜10px）
- ボタン・カード・入力欄すべて

### ボーダー・影
- 1px の極細ライン（rgba 0.1〜0.15）
- 影は極力使わない、使うなら 0 2px 8px rgba(0,0,0,0.04) 程度

### インタラクション
- ホバー: 背景色がわずかに変化
- トランジション: 200〜300ms ease-out
- 急な変化を避ける

### アイコン
- SF Symbols 風（lucide-react など線画系）
- 単色 / 1.5px ストローク

### NG事項
- グラデーション多用
- 強い影（ドロップシャドウ）
- 派手なアニメーション
- 角ばったボタン
- 太いボーダー
